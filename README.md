## Sobre o Projeto
Este é um sistema de rifa online desenvolvido com **Preact** no frontend e **Node.js/Prisma/Express** no backend. O objetivo é arrecadar fundos para me ajudar a comprar uma bicicleta e talvez um celular novo para os estudos.

---

## Como Começar

-   `npm run dev` - Inicia o servidor de desenvolvimento em http://localhost:5173/
-   `npm run server` - Inicia o backend da API com nodemon
-   `npm run dev:all` - Inicia frontend e backend simultaneamente
-   `npm run build` - Gera a build de produção em `dist/`
-   `npm run preview` - Testa a build de produção localmente em http://localhost:4173/

## Tecnologias Utilizadas
- **Frontend:** Preact, Vite, CSS Vanilla (Windows XP Style)
- **Backend:** Node.js, Express, Prisma (PostgreSQL)
- **Hospedagem:** Vercel
